Util

isMobile()

현재 클라이언트의 모바일 디바이스 여부를 리턴합니다.

true: mobile, false: pc

moc.util.isMobile()

getMobileDevice()

현재 모바일 디바이스 정보를 리턴합니다.

모바일 디바이스가 아닌경우 공백을 리턴합니다.

moc.util.getMobileDevice()

getBrowser()

현재 브라우저 정보를 리턴합니다.

moc.util.getBrowser()

formSerializeJsonObject()

Form의 input, select 등의 값을 Json Object로 리턴합니다.

moc.util.formSerializeJsonObject()

setCookie(name, value, date)

쿠키를 저장합니다. name, value 페어로 저장되며 date에 입력한 날자만큼 저장됩니다.

(date가 null일시 기본 30일)

setCookie('이름', '값', 30)
